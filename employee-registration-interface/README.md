# Employee Registration Interface

## Overview
This project is an Employee Registration Interface that allows users to register and log in. It features a professional design with a user-friendly layout.

## Project Structure
```
employee-registration-interface
├── css
│   └── styles.css
├── js
│   └── scripts.js
├── index.html
└── README.md
```

## Files Description
- **index.html**: The main HTML document that includes the welcome message, navigation bar, and links to the registration and login forms.
- **css/styles.css**: Contains styles for the interface, including layout, colors, fonts, and responsive design elements.
- **js/scripts.js**: Includes JavaScript functions for user interactions, such as form validation and navigation between forms.

## Features
- User registration with fields for first name, last name, email, password, and password confirmation.
- Login functionality with email and password fields.
- Navigation bar with links to various sections of the interface.
- Responsive design for optimal viewing on different devices.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open `index.html` in a web browser to view the interface.
3. Ensure that the CSS and JavaScript files are correctly linked for full functionality.

## Usage Guidelines
- Fill out the registration form to create a new account.
- Use the login form to access your account after registration.
- Navigate through the interface using the top navigation bar.
